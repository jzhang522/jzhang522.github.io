Deck Master

  • C#
  • JavaScript
  • SQL
  • Bootstrap
  • SQL Server

An ASP.NET Core MVC application enhanced with role-based authorization, secure user management, reCAPTCHA protection, and PayPal payment integration, including transaction tracking and access control.

DeckMaster is a full-stack ASP.NET Core MVC application focused on implementing secure authentication, role-based authorization, and payment processing. This project extends an existing MVC application by introducing administrative role management, protected user operations, and PayPal checkout functionality with persistent transaction tracking. Role-based access control was implemented to restrict sensitive features to authorized users. Only Admin and Manager roles can manage application roles and user permissions, ensuring separation of concerns and secure system administration. Navigation elements dynamically render based on user roles, preventing unauthorized access at both the UI and controller levels. To prevent automated sign-ups, Google reCAPTCHA was integrated into the registration workflow, improving application security and user validation.
